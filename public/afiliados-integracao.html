Configurações de Integração

Meta Pixel (Facebook Pixel)

Não configurado

Configure seu Meta Pixel ID para rastrear conversões de produtos afiliados. O rastreamento será ativado automaticamente no checkout e na página de sucesso de pagamento.

Digite apenas números (15 ou 16 dígitos). Você encontra este ID no Gerenciador de Eventos do Facebook.
Como encontrar seu Meta Pixel ID:
1. Acesse o Gerenciador de Eventos do Facebook
2. Selecione seu Pixel
3. O ID do Pixel está na parte superior da página (ex: 123456789012345)

UTMify

Não configurado

Configure seu API Token do UTMify para rastrear conversões de produtos afiliados. O rastreamento será ativado automaticamente no checkout e na página de sucesso de pagamento.

Seu token de API do UTMify. Mantenha este token seguro e não compartilhe com ninguém.
Como encontrar seu UTMify API Token:
1. Acesse sua conta no UTMify
2. Vá em Configurações > API
3. Copie seu API Token
