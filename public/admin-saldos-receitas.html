Detalhes das Estatísticas

Receita Disponível

Este valor representa o total de receita que os vendedores ainda podem sacar. É calculado como: Receita Total - Saques Processados

Receita Total

Este valor representa o total de vendas aprovadas de todos os vendedores no sistema.

Saldo ADM

Este valor representa o total de saques que foram processados e pagos aos vendedores. Este valor fica disponível para transferências B2C.

Transferências B2C

Este valor representa o total de transferências B2C realizadas (levantamentos) via Emola e M-Pesa.
